# Tech Stack: StudyMed

## Frontend
- **Framework**: React v19.1.0
- **Build Tool**: react-scripts
- **Styling**: Tailwind CSS v3.4.4
- **Routing**: react-router-dom v7.7.1
- **API Client**: Axios v1.11.0
- **Utils**: react-icons, uuid

## Backend
- **Framework**: Express v4.18.2
- **Runtime**: Node.js
- **Database**: MongoDB (via Mongoose v6.12.0)
- **Security**: cors (Auth/JWT đã loại bỏ)
- **File Upload**: multer, multer-storage-cloudinary, cloudinary

## System & Tools
- **Deployment**: Render (Server), Vercel (Client)
- **Dev Tools**: nodemon, react-scripts
